# Vanilla @mention
@mention plugin written in pure ES6 javascript.  
Minified JS: 2,31 KB. No dependencies.

[Give it a try on JS Fiddle](https://jsfiddle.net/nthishen/6ebspa2j/9/) 

# Quick Start
```
mention({
  id : 'textarea id here'
  symbol : 'optional. @ set as default'
  people : [{
    name : 'name1',
    username : 'username1',
    image : 'image url here'
  }]
})
```
