# Vanilla Mention
@ mention/tag plugin written in pure ES6 javascript.
Minified JS: 2,32 KB.

[View demo on JS Fiddle](https://jsfiddle.net/nthishen/6ebspa2j/1/) 

# Quick Start
```
mention({
  id : 'textarea id here'
  symbol : 'optional. @ set as default'
  people : [{
    name : 'name1',
    username : 'username1',
    image : 'image url here'
  }]
})
```
